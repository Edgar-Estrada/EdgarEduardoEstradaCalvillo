¿Qué es GitHub?

Es una herramienta que nos permite alojar el código de las aplicaciones de cualquier desarrollador. Su objetivo es que los desarrolladores suban el código de sus proyectos y que como usuario no sólo puedas descargarte la aplicación, sino también entrar al perfil del desarrollador para leer sobre él o ella y poder colaborar en su desarrollo.

Fundación

El software fue desarrollado por Chris Wanstrath, P. J. Hyett, Tom Preston-Werner y Scott Chancon usando Ruby on Rails. Antes del nombre GitHub Inc., era conocido como Logical Awesome LLC. El 4 de Junio de 2018 Microsoft compró GitHub por 7500 millones de dólares.
